# Fake Store API Flutter App

## Description

This Flutter application integrates the Fake Store REST API to fetch and display product data dynamically. The app uses HTTP requests and JSON parsing to show product images, titles, prices, and descriptions in a modern GridView layout.

## API Used

https://fakestoreapi.com/products

## Features

* REST API Integration
* JSON Parsing
* GridView UI
* Product Images
* Responsive Design
* Loading Indicator

## Technologies Used

* Flutter
* Dart
* HTTP Package

## Output


The application successfully fetches product data from the Fake Store REST API and displays it in a responsive GridView layout. Each product card shows the product image, title, price, and description dynamically using JSON parsing and HTTP requests in Flutter.


## How to Run

1. flutter pub get
2. flutter run
